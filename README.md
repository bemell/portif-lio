# Meu portfólio

Este projeto foi criado e pensado com intenção de aplicar um site responsivo pessoal como um portfólio. Ele foi desenvolvido com PHP, JavaScript, HTML e CSS.

Tem como objetivo apresentar minhas hablidades e evolução. 

## funcionalidades
-Apresentação pessoal
-Skills
-Formulário de contato

## tecnologia 
-HTML
-CSS
-JavaScript
-PHP
